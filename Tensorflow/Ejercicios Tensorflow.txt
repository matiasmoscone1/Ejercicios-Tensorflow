******************************* Sección 1: Tensores y operaciones *******************************

Crear un tensor 1D:
Crea un tensor de una sola dimensión con los números del 1 al 5 y muéstralo por consola.

Tensor 2D:
Crea un tensor de 2x2 con valores numéricos y obtén su forma (shape) con .shape.

Suma de tensores:
Crea dos tensores 1D con [1, 2, 3] y [4, 5, 6] y súmalos usando tf.add().

Multiplicación escalar:
Multiplica el tensor [1, 2, 3] por 3 usando tf.mul() o el operador equivalente.

Producto punto:
Calcula el producto punto entre [1, 2, 3] y [4, 5, 6] usando tf.dot().

Operaciones encadenadas:
Crea un tensor [2, 4, 6, 8], réstale 2 y luego divídelo por 2 (en una sola línea de código).

Transformar tensores:
Crea un tensor 2D de 3x2 y transpónlo (tf.transpose).

Convertir tensor a array:
Crea un tensor y conviértelo a un array JavaScript usando .array() o .dataSync().

Tensor aleatorio:
Genera un tensor de 3x3 con números aleatorios entre 0 y 1.

Normalización básica:
Crea un tensor [2, 4, 6, 8], calcula el valor máximo y mínimo, y normaliza los valores entre 0 y 1.

Modelo lineal simple:

Entrenamiento básico:
Entrena el modelo anterior con datos de entrada [1, 2, 3, 4] y salidas [1, 3, 5, 7] (una relación lineal simple y=2x−1).

Predicción:

Guardar y cargar modelo:
